Vector Space

Vector Space

the fundamental concepts of linear algebra are vector space. Vector spaces are sets of some elements statisfying some rules. Usually the vector space of most interest to us is $C^{n}$.

Vector

Let $z = (z_1, z_2, \cdots, z_n) \in C^{n}$, then $z$ is a vector. Basic operations like addition and multiplication are supported between vectors. A vector space must be closed under multiplication and addtion.

In Direct Notations, we usually use “$|\varphi\rangle$” to denote a vector (also called a “ket”).

Subspace

A vector subspace $W$ of a vector space $V$ is a subset of $V$, that is, $W$ also must be closed under scalar multiplication and addition.

The dimension of a vector space $V$ is the number of bases(which will be mentioned later) that be able to span $V$.

Bases and Linear Independence

Linear Independence

A set of non-zero vectors $|v_1\rangle, \cdots |v_n\rangle$ are linearly dependent iff there exists a set of complex numbers $a_1,\cdots,a_n$ with at least one $a_i \neq0$, such that$\sum a_{i} |v_{i} \rangle = 0$.

Otherwise, those vectors are linearly independent. A spaning set $|i\rangle$ of a vector space $V$ that are linearly independent forms a basis of $V$. (In this post we my denote a set by elements in it.)

Linear operater and Matrix

linear operator

a function $A$ mapping from vector space $V$ to vector space $W$ is a linear operator, if $A$ satisfies: $A(\sum_{i}a_i|v_i\rangle) =\sum_{i}a_iA(|v_i\rangle).$ Obvious, $(BA)|v_i\rangle=B(A|v_i\rangle)$.

There are some special linear operators. $I_V:V\rightarrow V$ is linear operator satisfies $I_V |v_i\rangle=|v_i\rangle$.

Matrix

A matrix is a set of elements with neatly arranged rows and columns. Matrix can implicate different meanings in different contexts.
It easy to verify that a matrix is a linear operator when applying matrix multiplication to matrix and vector. The connection between linear operator and matrix from other side can be established by the definition of matrix representing of an arbitrary linear operator $A:V\rightarrow W$:

$A|v_j\rangle =\sum_{i}A_{ij}|w_i\rangle$.

Where $A_{ij}$ is the element of $i$-th row and $j$-th column of matrix representing. Note that to make the connection between matrices and linear operators, we must
specify a set of input and output basis states for the input and output vector spaces of
the linear operator!

A notable fact is that when using same input base and output base, the matrix representing of an linear operator express exactly meaning as this matrix in context of matrix multiplication.

Pauli Matrix

$$\sigma_0=I=\left[ \begin{matrix} 1 & 0 \ 0&1 \end{matrix} \right]$$
$$\sigma_1=X=\left[ \begin{matrix} 0 & 1 \ 1&0 \end{matrix} \right]$$
$$\sigma_2=Y=\left[ \begin{matrix} 0 & -i \ i&0 \end{matrix} \right]$$
$$\sigma_3=Z=\left[ \begin{matrix} 1 & 0 \ 0&-1 \end{matrix} \right]$$
